# Instalación y configuración

Configurar Idiomas es muy sencillo: solo necesitas tu API key de Gemini y descargar la aplicación.

---

## 1. Obtener tu API Key de Gemini

**Qué es una API key?**  
Es tu clave personal para usar los modelos de Google Gemini. La aplicación la utiliza tanto para generar y evaluar el examen como para reproducir las voces del simulador. Dado que el uso de Gemini es de pago, puedes concebir a la API key como una contraseña y una tarjeta de crédito a la vez. 

### Crear la clave

1. Entra en [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Inicia sesión con tu cuenta de Google habitual.
3. Haz clic en **Create API key**.
4. Elige **Create API key in a new project**.
5. Copia la clave que aparecerá en pantalla y guárdala de forma permanente.

### Cosas importantes a tener en cuenta

- **Solo necesitas una clave:** la misma API key de Gemini sirve para generar preguntas, evaluar respuestas y reproducir las voces.
- **Es secreta y personal:** no compartas esta clave con nadie. Guárdala como si fuera la contraseña de tu correo y tu tarjeta de débito. 
- **Puedes cambiarla cuando quieras:** si en algún momento quieres generar una clave nueva, puedes hacerlo desde Google AI Studio y eliminar la anterior.

---

## 2. Descargar y preparar la aplicación

1. Entra en la [página de descargas del proyecto](https://github.com/aalonsogz/Idiomas/tags).
2. Haz clic en la versión más reciente.
3. En la sección **Assets**, descarga el archivo `.zip` de la aplicación.

### Preparar la carpeta y crear un acceso directo

1. Localiza el archivo `.zip` en tu carpeta de Descargas.
2. Haz clic derecho sobre el `.zip` y elige **Extraer todo**.
3. Abre la carpeta extraída y comprueba que dentro están `Idiomas.exe` y la carpeta interna de la aplicación.
4. Haz clic derecho sobre `Idiomas.exe`, crea un **acceso directo** y muévelo al Escritorio para tener la aplicación siempre a mano.

### Iniciar el simulador

1. Abre la app desde el acceso directo o desde `Idiomas.exe`.
2. La primera vez puede tardar unos 20-30 segundos en cargar. Es normal.

---

## Nota de seguridad de Windows

Al descargar o abrir la aplicación, Windows puede mostrar una advertencia azul diciendo "Windows protegió su PC" o "Smart App Control bloqueó la aplicación". Es normal en programas independientes que no llevan certificado de Microsoft.

Si aparece ese aviso:

1. Pulsa en **Más información**.
2. Luego pulsa en **Ejecutar de todas formas**.
3. A veces aparece un aviso y no da opción de usar la aplicación. Si te ocurre esto, espera 20-30 minutos después de haberla descargado por primera vez antes de volver a intentar usarla.
