# Instalacion y configuracion

Configurar Idiomas es muy sencillo: solo necesitas tu API key de Gemini y descargar la aplicacion.

---

## 1. Obtener tu API Key de Gemini

**Que es una API key?**  
Es tu clave personal para usar los modelos de Google Gemini. La aplicacion la utiliza tanto para generar y evaluar el examen como para reproducir las voces del simulador.

### Crear la clave

1. Entra en [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Inicia sesion con tu cuenta de Google habitual.
3. Haz clic en **Create API key**.
4. Elige **Create API key in a new project**.
5. Copia la clave que aparecera en pantalla y guardala de forma permanente.

### Cosas importantes a tener en cuenta

- **Solo necesitas una clave:** la misma API key de Gemini sirve para generar preguntas, evaluar respuestas y reproducir las voces.
- **Es secreta y personal:** no compartas esta clave con nadie. Guardala como si fuera la contrasena de tu correo.
- **Puedes cambiarla cuando quieras:** si en algun momento quieres generar una clave nueva, puedes hacerlo desde Google AI Studio y eliminar la anterior.

---

## 2. Descargar y preparar la aplicacion

1. Entra en la [pagina de descargas del proyecto](https://github.com/aalonsogz/Idiomas/tags).
2. Haz clic en la version mas reciente.
3. En la seccion **Assets**, descarga el archivo `.zip` de la aplicacion.

### Preparar la carpeta y crear un acceso directo

1. Localiza el archivo `.zip` en tu carpeta de Descargas.
2. Haz clic derecho sobre el `.zip` y elige **Extraer todo**.
3. Abre la carpeta extraida y comprueba que dentro estan `Idiomas.exe` y la carpeta interna de la aplicacion.
4. Haz clic derecho sobre `Idiomas.exe`, crea un **acceso directo** y muevelo al Escritorio para tener la aplicacion siempre a mano.

### Iniciar el simulador

1. Abre la app desde el acceso directo o desde `Idiomas.exe`.
2. La primera vez puede tardar unos 20-30 segundos en cargar. Es normal.

---

## Nota de seguridad de Windows

Al descargar o abrir la aplicacion, Windows puede mostrar una advertencia azul diciendo "Windows protegio su PC" o "Smart App Control bloqueo la aplicacion". Es normal en programas independientes que no llevan certificado de Microsoft.

Si aparece ese aviso:

1. Pulsa en **Mas informacion**.
2. Luego pulsa en **Ejecutar de todas formas**.
